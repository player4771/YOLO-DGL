## This repository is the official implementation of the paper: "YOLO-DGL: A Tea Disease Detection Model Based on Improved YOLOv13".

### Install
```bash
git clone https://github.com/player4771/YOLO-DGL.git
```
#### The configuration file of YOLO-DGL is located at models/YOLO/ultralytics/cfg/models/v13/yolov13_edit13.yaml
#### yolov13_edit13_{1,2,3,4}.yaml are the configuration files of the comparison models used in the ablation experiments.
